User Study Document of KGVis

1、Task one

Drag the border of Karl_Marx, then double click the edge, and input predicate 'influencedBy'.

After this you will get a list of result. you can choose expand result while you right click the result ring.

2、Task two

Add a new node through the top button, double click the new node and input 'Bob_Black'.

Drag the border of Bob_Black and drag to the last result node, as shown in the following figure.

Double click the new edge, and input predicate 'influencedBy'. The number of the result node is change.

3、Task three

Query the people who influenced Karl_Marx and Bob_Black, influenced by who.

4、Task four

Add a filter operation on the result of task three.

Right click the result and choose filter operation, then input 'birthDate>1750'

Then you will find that the results have changed.
